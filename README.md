## Getting Started

To get a local copy up and running follow these simple steps.

1. To get started, fork this repository to your GitHub account.

2. Clone the repo.
    ```sh
     git clone https://github.com/<your-username>/landing-page.git
    ```
3. Install packages.
    ```sh
     npm install
    ```
4. Run project.
    ```sh
     npm start
    ```
